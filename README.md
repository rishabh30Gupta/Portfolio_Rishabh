# Rishabh Gupta - Portfolio Website

A modern, animated portfolio website showcasing the work and skills of Rishabh Gupta, a Software Engineer and DevOps specialist.

## Features

- Royal blue and white theme design
- Smooth scroll animations and parallax effects
- Responsive design for all devices
- Project showcase with GitHub links
- Experience timeline
- Skills and achievements sections
- Contact information

## Tech Stack

- React + TypeScript
- Vite
- CSS3 with custom animations
- GSAP for advanced animations

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # React components
├── data/          # Portfolio data (content)
├── theme/         # Theme configuration
├── hooks/         # Custom React hooks
└── context/       # React context providers
```

## Customization

To customize the portfolio content, edit `src/data/portfolioData.ts`.

To modify the theme colors, update `src/theme/theme.ts` and `src/index.css`.
